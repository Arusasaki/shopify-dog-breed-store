# 【超初心者向け】「わんこのおみせ」構築ガイド

これは**「中学生でもできる」**を目標にした、わかりやすい手順書です。
難しいプログラミングの知識は一切いりません。「コピー＆ペースト（貼り付け）」ができれば大丈夫です！

---

## 全体の流れ
私たちがやろうとしていることは、たったの3ステップです。
1.  **「箱」を作る**: Shopifyに「犬種のロゴ」や「テーマカラー」を入れるための新しい**入力欄**を作ります。
2.  **「魔法」をかける**: お客さんがそのページに来たら、自動的にロゴが変わるように、プログラムをコピペします。
3.  **「中身」を入れる**: 実際に「柴犬のロゴ」や「プードルの色」を登録します。

---

## 準備するもの
*   パソコン
*   Shopifyの管理画面（ログインしておいてください）
*   このフォルダに入っている `project_assets` というフォルダの中身

---

## ステップ 1: 入力欄（箱）を作ろう

まずは、Shopifyに「このコレクション（犬種）には、このロゴを使ってね」と教えるための設定場所を作ります。これがないと、Shopifyはどこに画像を保存していいかわかりません。

1.  Shopifyの管理画面の左下にある **「設定」**（歯車のマーク）をクリックします。
2.  左のメニューから、下の方にある **「メタフィールドとメタオブジェクト」**（または「カスタムデータ」）をクリックします。
3.  出てきたリストの中から **「コレクション」** をクリックします。
4.  右上の **「定義を追加」** という緑色のボタンをクリックします。
5.  ここで、**1つ目の箱（ロゴ用）** を作ります。以下の通りに入力してください。
    *   **名前**: `Brand Logo` と入力
    *   **ネームスペースとキー**: ここが重要です！ `custom.brand_logo` となっているか確認してください。（自動で入ることもありますが、必ず確認！）
    *   **説明**: 空欄でOK
    *   **タイプを選択**: 「ファイル」を選びます。そのあと「画像のみ」にチェックが入ってればOKです。
    *   最後に **「保存」** をクリック。
6.  もう一度 **「定義を追加」** を押して、**2つ目の箱（色用）** を作ります。
    *   **名前**: `Brand Color Primary` と入力
    *   **ネームスペースとキー**: `custom.brand_color` （ここ大事！）
    *   **タイプを選択**: 「色」を選びます。
    *   **「保存」** をクリック。
7.  もう一度 **「定義を追加」** を押して、**3つ目の箱（バナー画像用）** を作ります。
    *   **名前**: `Hero Banner` と入力
    *   **ネームスペースとキー**: `custom.hero_banner` （ここ大事！）
    *   **タイプを選択**: 「ファイル」を選びます。
    *   **「保存」** をクリック。

これで「箱」の準備は完璧です！設定画面を閉じてください。

---

## ステップ 2: 魔法のコードを貼り付けよう

ここが一番難しそうに見えますが、**ただのコピペ作業**です。深呼吸してやってみましょう。

### A. ロゴが変わる魔法
1.  `sections/header.liquid` を開きます。
2.  `Ctrl + F` (Macなら `Command + F`) で、以下の文字を検索してください。
    `<a href="{{ routes.root_url }}" class="header__heading-link`
3.  この行のすぐ下に、以下のようなコードの塊があるはずです（**ファイルの中に2箇所あります**）。
    
    ```liquid
    {%- if settings.logo != blank -%}
      <div class="header__heading-logo-wrapper">
        ... (中身はいろいろ) ...
      </div>
    {%- else -%}
      <span class="h2">{{ shop.name }}</span>
    {%- endif -%}
    ```

4.  **この `{%- if ...` から `{%- endif -%}` までの塊を丸ごと削除して、**代わりに `project_assets/header_logic.liquid` の中身を貼り付けます。
5.  **重要:** 同じコードがファイルの下の方（160行目付近）にもう一度出てきます。**2箇所とも同じように書き換えてください。**
6.  右上の **「保存」** をクリック。

### B. 「世界一のUI」にする魔法（デザイン刷新）
1.  同じ画面の左側で、今度は `theme.liquid` というファイルを探します。一番上の **「Layout」** というフォルダに入っていることが多いです。
2.  ファイルを開いたら、一番下までスクロールしてください。ずーっと下です。
3.  下から数行目に `</head>` という文字があるはずです。終わりの合図です。
4.  この `</head>` の**直前の行**に隙間を作って、フォルダ内の `project_assets/world_class_design.liquid` の中身を**全部コピーして貼り付けます**。
    *   *以前の、少しだけ色が変わるコードではありません。今回はサイト全体を高級なデザインに変える「特別なコード」です！*
5.  右上の **「保存」** をクリック。

これで難しい作業はすべて終了です！お疲れ様でした！「コードを編集」画面を閉じてください。

---

## ステップ 3: 実際に犬種を設定してみよう

さあ、楽しい時間の始まりです。実際に「柴犬のお店」を作ってみましょう。

1.  管理画面の左メニューから **「商品管理」** をクリックし、その下の **「コレクション」** を選びます。
2.  右上の **「コレクションを作成」** をクリック。
3.  タイトルに「柴犬」と入力します。
4.  （重要）ページの一番下までスクロールしてください。
5.  **「メタフィールド」** という項目ができているはずです！さっき作った「箱」です。
    *   **Brand Logo**: ここに、Geminiで作った「柴犬ショップのロゴ画像」をアップロードします。
    *   **Brand Color Primary**: ここをクリックして、柴犬っぽい色（例えば茶色）を選んでください。
6.  **「保存」** をクリック。

---

## 完成！確認してみよう

1.  今作った「柴犬」コレクションの画面の上に **「表示」** というボタンがあるのでクリックします。
2.  お店のページが開きます。
3.  **見てください！**
    *   左上のロゴが、いつものお店のロゴではなく「柴犬のロゴ」になっていませんか？
    *   ボタンや線の色が、あなたが選んだ「茶色」になっていませんか？
4.  次に、お店のトップページに戻ってみてください。
    *   普通のロゴと、普通の色に戻っていれば大成功です！

### 次にやること

*   商品は Printful で作って、それぞれのコレクションに追加していけばOKです。
